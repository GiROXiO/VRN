# VRN - Teoría de Códigos

Aplicación de escritorio para el análisis y construcción de códigos lineales,
desarrollada como proyecto final del curso de Teoría de Códigos.

## Tecnologías
- **Frontend:** C# con WPF (.NET 10)
- **Backend:** Python (FastAPI)

## Requisitos
- .NET 10 SDK
- Python 3.x

## Cómo ejecutar
1. Crear el entorno virtual: `python -m venv venv`
2. Activar el entorno virtual: `venv\Scripts\activate`
3. pip install fastapi uvicorn numpy sympy
4. Iniciar el servidor: `uvicorn api:app --reload`
5. Ejecutar la app: `dotnet run`

## Puntos implementados
- **P1:** Código Reed-Solomon sobre GF(q)
- **P2:** Código lineal binario y operaciones
- **P3:** Código cíclico binario
